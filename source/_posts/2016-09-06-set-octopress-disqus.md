Disqus 申請帳號

登入Disqus帳號

點選 右上頭像 > 點選 Settings

點選 右上頭像 > 點選 Add Disqus To Site

點選 Start Using Engage

Site name 輸入留言板的名字

如: `copy0401`

Choose your unique Disqus URL 輸入留言版的網址

如: http://disqus.com/by/copy0401

Category 選擇類別

如: Tech


```
cd ~/octopress
vim _config.yml
disqus_short_name: copy0401
disqus_show_comment_count: true
```
